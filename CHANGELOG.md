# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-17

### Added
- Initial release of `grapesjs-react-renderer`
- Core plugin functionality for rendering React components in GrapesJS canvas
- Full TypeScript support with generics and strict typing
- Support for complex component props via JSON traits
- Custom prop mapping via `mapProps` function
- React Provider integration (Redux, Context, custom providers)
- Factory functions for common component patterns
- Helper utilities for trait parsing and prop transformation
- Comprehensive documentation and examples

### Features
- Simple component configuration
- UI Component pattern support
- Custom prop mapping
- Provider wrapping (single, multiple, custom trees)
- Type-safe component registration
- JSON trait support for complex data structures
